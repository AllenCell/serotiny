serotiny.io.dataframe.dataframe_dataset module¶

class serotiny.io.dataframe.dataframe_dataset.DataframeDataset(dataframe, loaders=None, iloc=True)[source]¶

Bases: Generic[torch.utils.data.dataset.T_co]

Class to wrap a pandas DataFrame in a pytorch Dataset. In practice, at AICS we use this to wrap manifest dataframes that point to the image files that correspond to a cell. The loaders dict contains a loading function for each key, normally consisting of a function to load the contents of a file from a path.

Parameters

  • dataframe (pd.DataFrame) – The file which points to or contains the data to be loaded

  • loaders (dict) – A dict which contains methods to appropriately load data from columns in the dataset.
